# Alfred Search Route Workflow
Alfred PowerPackを購入したので、習作としてGoogleMapの経路探索ができるWorkflowを作りました。
AlfredのWeb SearchやCustom URLだと2つ以上のqueryを入力させることができないっぽかったので、bash scriptを使って書きました。
